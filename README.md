AI Sales Calling & Slot Booking System

Overview

AI Sales Calling & Slot Booking System is an intelligent voice automation application that makes outbound sales calls, answers customer queries, and books appointment slots through natural conversations.

The system uses Generative AI to understand user responses and automate the complete sales calling workflow.

⸻

Features

* AI-powered voice conversations
* Automated outbound calling
* Intelligent customer query handling
* Appointment / Slot booking
* Real-time conversational workflow
* Easy configuration using environment variables

⸻

Tech Stack

* Python
* Flask
* Google Gemini AI
* Twilio Voice API
* HTML
* CSS
* JavaScript

Installation

git clone https://github.com/Subbuposina03/AI-Sales-Calling-System.git
cd AI-Sales-Calling-System
pip install -r requirements.txt

Create a .env file and configure:

* Gemini API Key
* Twilio Account SID
* Twilio Auth Token
* Twilio Phone Number
* Ngrok URL

Run:

python app.py
